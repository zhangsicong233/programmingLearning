# 二分答案

**最大化答案(以 int 为例)**

```
check(int x) {
  ...   // 计算 y

  return y <= C  // 当 x 增大时，y 随之增大, 即增函数
  return y >= C  // 当 x 增大时，y 随之减小, 即减函数
}
```

```
binarySearchAns() {
  int l = 下界 - 1；
  int r = 上界 + 1;
  while (l + 1 < r) {
    int mid = (l + r) / 2;
    if (check(mid)) {
      l = mid;  // 最大化 
    } else {
      r = mid;
    }
  }
}
```
