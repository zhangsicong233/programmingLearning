# 高精度除法(高精度 / 低精度`)

被除数(高精度) `a` 和除数(低精度) `b` ;

## 高精度除法实现

```
void highPrecisionDiv(const std::vector<int>& x, const int& y,
                      std::vector<int>& res) {
  long long temp = 0;
  for (int i = x.size() - 1; i >= 0; --i) {
    temp = temp * 10 + x[i];
    res.push_back(temp / y);
    temp %= y;
  }

  std::reverse(res.begin(), res.end());

  // 去除多余的 0
  while (res.size() > 1 && !res.back()) {
    res.pop_back();
  }
}
```
