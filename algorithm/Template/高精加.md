# 高精加

加数 `a` 和 `b` ;

## 高精度加法实现

```
void highPrecisionAdd(int* x, int* y, int* res) {
  for (int i = 0; i < resLen; ++i) {
    res[i] += x[i] + y[i];     // 若无进位，初始为 0， 有进位，初始为 1
    res[i + 1] = res[i] / 10;  // 处理进位
    res[i] %= 10;
  }

  if (res[resLen]) {  // 若有进位，则将长度扩大 1
    ++resLen;
  }
}
```

## 预处理

通过把字符串翻转，将数字的低位存入数组

```
for (int i = xLen - 1; i >= 0; --i) {
  x[xLen - 1 - i] = a[i] - '0';
}

for (int i = yLen - 1; i >= 0; --i) {
  y[yLen - 1 - i] = b[i] - '0';
}
```

> 举例说明：
> 当数字为： 153
> 所得字符串为："153"，则数字的最低位 3 的下标为 2，需要通过翻转将下标转换成 0 并存入 int 数组中
