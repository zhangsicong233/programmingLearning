# 高精度加法

加数 `a` 和 `b` ;

## 高精度加法实现

```
void highPrecisionAdd(std::vector<int>& x, std::vector<int> y, std::vector<int>& res) {
  int temp = 0;
  for (int i = 0; i < x.size() || i < y.size(); ++i) {
    if (i < x.size()) {
      temp += x[i];
    }
    
    if (i < y.size()) {
      temp += y[i];
    }

    res.push_back(temp % 10);  // 存余
    temp /= 10;  // 处理进位
  }

  if (temp) {  // 若有进位，将进位存入数组
    res.push_back(temp);
  }
}
```

## 预处理

通过把字符串翻转，将数字的低位存入数组

```
for (int i = a.size() - 1; i >= 0; --i) {
  x.push_back(a[i] - '0');
}

for (int i = b.size() - 1; i >= 0; --i) {
  y.push_back(b[i] - '0');
}
```

> 举例说明：
> 当数字为： 153
> 所得字符串为："153"，则数字的最低位 3 的下标为 2，需要通过翻转将下标转换成 0 并存入 int 数组中
