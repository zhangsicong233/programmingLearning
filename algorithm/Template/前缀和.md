# 前缀和

**前缀和数组通常下标从 1 开始**

## 一维前缀和

公式：S<sub>i</sub> = S<sub>i - 1</sub> + a<sub>i</sub>

求区间 [l, r] 之间的值便可以直接使用维护好的前缀和数组，通过 S<sub>r</sub> - S<sub>l - 1</sub> 得到

## 二维前缀和

公式：S<sub>i, j</sub> = S<sub>i, j - 1</sub> + S<sub>i - 1, j</sub> - S<sub>i - 1, j - 1</sub> + a<sub>i, j</sub>

求由 (x1, y1) 到 (x2, y2) 所包围的区域的值便可以直接用维护好的二维前缀和数组通过 S<sub>x<sub>2</sub>, y<sub>2</sub></sub> - S<sub>x<sub>1</sub>-1, y<sub>2</sub></sub> - S<sub>x<sub>2</sub>, y<sub>1</sub>-1</sub> + S<sub>x<sub>1</sub>-1, y<sub>1</sub>-1</sub> 得到
