# 高精度乘法

乘数 `a` 和 `b` ;

## 高精度乘法实现

```
void highPrecisionMul(const std::vector<int>& x, const std::vector<int>& y,
                      std::vector<int>& res) {
  for (int i = 0; i < x.size(); ++i) {
    for (int j = 0; j < y.size(); ++j) {
      res[i + j] += x[i] * y[j];
      res[i + j + 1] += res[i + j] / 10;
      res[i + j] %= 10;
    }
  }

  // 去除前导零
  while (res.size() > 1 && !res.back()) {
    res.pop_back();
  }
}
```

## 预处理

预先分配空间，避免越界

```
res.resize(x.size() + y.size(), 0);
```
